## Sections of a `RMS` Script
```
1. <PLAYER_SETUP>
2. <LAND_GENERATION>
3. <ELEVATION_GENERATION>
4. <CLIFF_GENERATION>
5. <TERRAIN_GENERATION>
6. <OBJECTS_GENERATION>
```
Sections are not required to be in order, and not all sections are required for map generation.
